# Рекреационная инфраструктура Краснодара
## Парки и озеленённые территории

Источник данных: [OSM](https://www.openstreetmap.org/) (административные границы, озеленённые территории, жилые здания).

Методология:

1. выгружены все полигональные объекты «зелёных» территорий и жилых зданий;
2. отфильтрованы только озеленённые терриории, пригодные для рекреации;
3. на основе средней обеспеченности жилой площадью в Краснодаре (44 кв.м/чел.) рассчитано приблизительное население каждого дома;
4. по гексагональной сетке рассчитана обеспеченность населения озеленёнными территориями;
5. построен heatmap по центроидам озеленённых территорий;
6. итоговые слои экспортированы через qgis2web, файл index.html расширен;
7. все файлы выложены на GitHub и «задеплоены».

**Что было самым сложным в работе?** Десятки раз экпортировать веб-карту через плагин и не понимать, почему ничего не отображается в браузере.

**Что получилось, чем мы можем гордиться?** Первым задеплоенным проектом на GitHub!

**Что не получилось, что можно было бы добавить?** Более качественную работу с расчётной численностью населения (через ГИС ЖКХ).

**Как мы использовали нейросети?** Для редактирования HTML-файла.
